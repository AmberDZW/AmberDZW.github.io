# 隐私与反馈说明

## 适合谁

适合第一次打开资料库的家长、老师或协作者。  
这页先说明：资料怎么拿、反馈怎么给、哪些信息不要填写。

## 先记住 3 句话

反馈只写资料问题，不写孩子信息。  
不要上传截图、作业原图、试卷、成绩单或聊天记录。  
学生不要自己填，让家长代填。

## 安全边界

- 资料可以直接查看或下载，不需要先填写表单。
- 反馈完全自愿，不作为领取资料的条件。
- 未成年人不要自行填写反馈，建议由家长代填。
- 不填写学生姓名、学校、班级、电话、地址、账号、成绩、家庭收入或照片。
- 不上传作业原图、试卷、成绩单、聊天截图、孩子照片或含学校班级信息的图片。

## 先说结论

反馈入口只用来发现资料哪里不清楚、哪里不安全、哪里需要更新。  
它不是私信答疑入口，也不是资料领取门槛。

## 可以反馈什么

| 可以反馈 | 示例 |
|---|---|
| 资料看不懂 | 哪个步骤不清楚、哪个词不明白 |
| 工具不可用 | 哪个入口打不开、是否出现付费或登录问题 |
| 安全风险 | 是否差点上传隐私、是否看到 AI 给出可直接提交的答案 |
| 下载问题 | 文件打不开、手机阅读困难、页面太大 |
| 下一版需求 | 希望增加哪类清单、模板或说明 |

安全反馈示例：

> 我用手机打开 Prompt 卡片时，可以复制，但 P02 的隐私检查不够明确。建议加一句“先不要粘贴原文，删掉隐私后再检查”。

## 不要反馈什么

| 不要填写 | 为什么 |
|---|---|
| 学生姓名、学校、班级 | 这些是敏感身份信息 |
| 电话、地址、账号 | 资料库不需要联系用户个人 |
| 家庭收入、家庭详细情况 | 项目不做身份筛选，不需要收集 |
| 学生照片、作业原图、成绩单 | 可能暴露未成年人隐私 |
| 整段聊天记录或老师批注 | 可能包含他人信息 |

## 建议反馈字段

反馈表控制在 1 分钟内完成。

| 字段 | 类型 | 是否必填 |
|---|---|---|
| 你的使用身份：家长 / 老师 / 其他 | 单选 | 可选 |
| 主要用手机还是电脑看资料 | 单选 | 可选 |
| 哪份资料最有用 | 多选 | 可选 |
| 哪一步最难 | 多选 | 可选 |
| 是否遇到工具不可用、登录或付费提示 | 多选 | 可选 |
| 是否发现 AI 答案错误或不安全情况 | 开放题 | 可选 |
| 下一版最希望增加什么 | 开放题 | 可选 |

学生反馈请由家长代填。来源渠道可以留空，不需要填写个人联系方式。

## 评论和私信怎么处理

- 如果评论里出现姓名、学校、照片、电话等信息，建议提醒删除。
- 如果有人发具体作业题，不在评论区给答案，只引导看“作业边界卡”。
- 如果有人请求一对一辅导，统一说明项目不做私信答疑。
- 高频问题会整理成公开资料，方便更多家庭一起看。
