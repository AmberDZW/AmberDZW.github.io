安装到手机

一个网址，苹果安卓都能用。

ZHIYU 是手机优先的网页 App。iPhone 不需要 APK，安卓也不一定要装包：打开网址后添加到主屏幕，就会像一个轻量 App 一样使用。

先打开工具箱 安卓 APK 备用包

如果浏览器支持一键安装，这里会出现安装按钮。

iPhone

  1. 用 Safari 打开 ZHIYU 网站，不要用微信内置浏览器。
  2. 点底部分享按钮。
  3. 选择“添加到主屏幕”。
  4. 桌面出现 ZHIYU 图标后，以后直接点图标进入。

安卓

  1. 用 Chrome、Edge 或系统浏览器打开 ZHIYU 网站。
  2. 如果页面出现“一键安装”，直接点击。
  3. 如果没有按钮，点浏览器菜单，选择“添加到主屏幕”或“安装应用”。
  4. 需要离线包时，再使用安卓 APK 备用包。

日常推荐使用网页 App：它跨平台、更新快、不会让用户反复下载安装。APK 只作为安卓备用方案；iPhone 用户走 Safari 添加主屏幕。
